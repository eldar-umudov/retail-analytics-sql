# Retail Analytics (SQL)

## Business context
Аналитический проект для розничного бизнеса в сегменте мебели и предметов интерьера.

Цель проекта — показать, как с помощью SQL и BI-инструментов можно анализировать продажи,
ассортимент и клиентское поведение для поддержки управленческих решений.

Данные основаны на реальной структуре бизнеса, обезличены и модифицированы.
Логика анализа и бизнес-смысл сохранены.

---

## Business questions
- Какие товары формируют основную выручку бизнеса?
- Насколько стабилен спрос по различным товарным группам?
- Какие клиенты приносят наибольшую ценность?
- Как меняется поведение клиентов со временем (повторные покупки)?
- Как сезонность влияет на продажи?
- Как скидки влияют на выручку и прибыль?

---

## Data model
Проект использует следующие сущности:

- `sales` — данные о продажах (дата, сумма, скидка)
- `customers` — клиенты
- `products` — товары
- `categories` — товарные категории
- `discounts` — акции и скидки

Данные представлены в виде реляционной модели и предназначены для аналитических запросов.

## Tools
- **MySQL** — аналитические SQL-запросы
- **SQL** — CTE, оконные функции, агрегации
- **Excel** — предварительный анализ и проверка гипотез
- **Power BI** — визуализация результатов (в отдельном репозитории)

## Analysis
В рамках проекта реализованы следующие виды анализа:

- ABC / XYZ анализ ассортимента
- RFM-анализ клиентов
- Cohort Analysis (повторные покупки)
- Анализ сезонности продаж
- Анализ среднего чека
- Анализ эффективности скидок

Все расчёты выполнены на уровне SQL и ориентированы на бизнес-задачи.

---

## Key insights
Примеры инсайтов, которые можно получить в рамках проекта:

- небольшая доля товаров формирует основную часть выручки
- товары с нестабильным спросом требуют особого управления запасами
- значительная часть клиентов совершает только одну покупку
- повторные покупки имеют выраженную зависимость от времени
- сезонность существенно влияет на объём продаж
- не все скидки положительно влияют на прибыль

---

## Business recommendations
На основе анализа могут быть приняты следующие решения:

- фокус на поддержание ассортимента группы A/X
- оптимизация или вывод товаров группы C/Z
- разработка программ удержания клиентов
- корректировка скидочной политики
- планирование закупок с учётом сезонности
